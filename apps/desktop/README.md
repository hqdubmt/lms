# MasterLMS Desktop

Ứng dụng desktop cho giảng viên và học viên sử dụng MasterLMS.

## Cách sử dụng

Lần đầu mở app, nhập địa chỉ máy chủ LMS của trường (ví dụ: `http://192.168.1.100:3000`).

## Build

```bash
cd apps/desktop
npm install

# Windows
npm run build:win

# macOS
npm run build:mac

# Linux
npm run build:linux
```

Kết quả trong thư mục `dist/`.

## Phát triển

```bash
npm start
```
